# 1. Project Overview

This project involves the development of "KnowCS," an interactive web-based platform specifically designed for the HKUST COMP 2211 (Introduction to Artificial Intelligence) course. With modern web technologies, the project aims to transform abstract computer science and artificial intelligence concepts into intuitive, interactive visual experiments. The goal is to facilitate a "learning-by-doing" approach, helping students deepen their understanding of complex models through direct engagement.

# 2. Project Objectives

The primary objective is to evolve the project from a collection of "web widgets" into a comprehensive Interactive Learning System. This platform will serve as a "Dynamic Lab Manual" for COMP 2211. When students encounter abstract formulas in linear algebra, clustering, or neural networks, they can access corresponding modules to perform hands-on operations. Through this "What You See Is What You Get" (WYSIWYG) interaction, static lecture notes are transformed into experiments where students can adjust parameters in real-time to observe algorithmic changes and grasp the underlying mathematical logic of AI. 

Technically, I am exploring the implementation of a "ready-to-use" Python online runtime environment directly within the browser. The intent is to allow students to experience the actual "model training" process rather than just viewing static results. Users will be able to observe (and potentially train) simple Large Language Models (LLMs) or Image Recognition models, witnessing how a model evolves from "zero knowledge" to successfully recognizing digits or generating text. 

In the long term, KnowCS is envisioned as a centralized hub for AI/CS extended learning resources and general academic tools. Planned integrations include interactive interpretations of classic papers, Manim mathematical animations, and Netron model topology analysis. By bridging with the "Knowit" project, KnowCS aims to become a cross-disciplinary resource center, fostering a sustainable, community-driven teaching model that can eventually be extended to other foundational courses such as COMP 1023.

# 4. Proposed Features & Technical Implementation

The project will align closely with the COMP 2211 syllabus [https://course.cse.ust.hk/comp2211/], with modules developed systematically to cover clustering, Naive Bayes, backpropagation, gradient descent, KNN, and NumPy operations. Beyond these core modules, I plan to develop an automated video subtitle script to enhance the accessibility of lecture recordings. If feasible, the system will also incorporate Netronstyle network architecture analysis and Manim-based educational videos for distribution on platforms like YouTube and Bilibili. 

To foster a deeper learning community, I will establish an AI technical blog to document development insights and challenges. A dedicated Classic AI Papers section is also planned to visualize landmark research such as Attention is All You Need, ResNet, and the DeepSeek-V3 Technology Report. This module will focus on modularizing complex formulas and making paper concepts intuitive through interactive interfaces. 

Additionally, the platform will explore HKUST authentication and automated synchronization with official course resources, transitioning KnowCS from an independent tool into a professional laboratory prototype.
